# 基本规范
每个问题与记录单独放置文件夹,不得直接放在子文件夹里,文件夹名基本格式为[{问题名_作者}]可自行按需添加时间与其他需要同样以_分割但不得超过512字符
文件夹基本格式
    这是一个问题_曙光
        main.md 文档和其相关的文章的主入口
        images 图片资源
        resource 代码等资源
        readme.md 基本说明文件
可以添加结构后再readme.md说明但不能少于基本结构(除非你不附录图片资源或代码资源)
